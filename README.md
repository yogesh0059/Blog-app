# Blog Management REST API

A RESTful Blog Management API built using Spring Boot. The application allows users to manage blog posts, categories, and comments with CRUD operations, validation, and proper exception handling.

## Features

- Create, update, delete, and view blog posts
- Manage blog categories
- Add and manage comments
- RESTful API endpoints
- Request validation
- Global exception handling
- Database integration using Spring Data JPA
- API testing with Postman

## Technologies Used

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- Hibernate
- H2 / PostgreSQL
- Maven
- Postman

- POST - Create new data
- PUT - Update existing data
- DELETE - Remove data
